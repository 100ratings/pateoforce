# Pateo Force Trainer

Um simulador simples para treinar a técnica de **Pateo Force** em mágica e mentalismo.

## Como Funciona

1. **Número Aleatório**: O jogo gera um número entre 6 e 20
2. **Sua Decisão**: Você escolhe se o espectador deve começar apontando ou eliminando
3. **Feedback**: O sistema diz se você acertou ou errou
4. **O Jogo**: Se acertar, você joga contra o computador (que age como espectador)

## A Regra de Ouro

- **PAR** (6, 8, 10...): Espectador começa **APONTANDO**
- **ÍMPAR** (7, 9, 11...): Espectador começa **ELIMINANDO**

## Como Usar

Abra o arquivo `index.html` no navegador. É só isso!

## Deploy no GitHub Pages

1. Crie um repositório no GitHub
2. Faça upload dos arquivos (apenas `index.html` é necessário)
3. Vá para Settings → Pages → Deploy from a branch
4. Selecione a branch `main` e pasta `/ (root)`
5. Pronto! Seu site estará em `https://seu-usuario.github.io/pateo-force-trainer`
