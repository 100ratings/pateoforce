# Pateo Force Trainer

Um simulador simples para treinar a técnica de **Pateo Force** em mágica e mentalismo.

## Como Funciona

1. **Pergunta 1**: O sistema escolhe aleatoriamente se o espectador ou mágico começa
2. **Quadradinhos**: Mostra visualmente a quantidade de itens (você conta)
3. **Pergunta 2**: Escolha entre "ESPECTADOR APONTA" ou "MÁGICO APONTA"
4. **Feedback**: Diz se acertou ou errou + mostra a quantidade em números

## A Regra de Ouro

- **PAR** (6, 8, 10...): Espectador começa **APONTANDO**
- **ÍMPAR** (7, 9, 11...): Mágico começa **APONTANDO**

## Como Usar

### No Navegador
Abra o arquivo `index.html` no navegador. É só isso!

### Como App (iOS/Android)
Para uma experiência fullscreen sem a barra de endereço:

**iOS:**
1. Abra o arquivo no Safari
2. Clique no botão "Compartilhar" (ícone de caixa com seta)
3. Selecione "Adicionar à Tela de Início"
4. Confirme

**Android:**
1. Abra o arquivo no Chrome
2. Clique no menu (três pontos)
3. Selecione "Instalar app"
4. Confirme

## Deploy no GitHub Pages

1. Crie um repositório no GitHub
2. Faça upload dos arquivos (`index.html`, `manifest.json`, `README.md`)
3. Vá para Settings → Pages → Deploy from a branch
4. Selecione a branch `main` e pasta `/ (root)`
5. Pronto! Seu site estará em `https://seu-usuario.github.io/pateo-force-trainer`

## Arquivos

- `index.html` - Aplicação completa (HTML + CSS + JavaScript)
- `manifest.json` - Configuração para instalar como app
- `README.md` - Este arquivo
